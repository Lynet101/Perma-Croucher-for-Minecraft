Description:
This script will toggle the shift key when "F1" is pressed, only if the process "Javaw.exe" is the current showing window,
to prevent interferance with the rest of the system. 

Disclaimer:
This script is originally intended for minecraft, But interferrance with other java based programs (such as geogebra) may occur. 
If this happens, please exit the program, and only launch it on game start.
